# Angular Hands-On — Student Course Portal

Single Angular 20 standalone project covering the 10 Digital Nurture hands-ons: components/binding, lifecycle and communication, directives/pipes, template and reactive forms, services/DI, routing/guards, HTTP/interceptors, NgRx and testing scaffolding.

## Run

```bash
npm install
npm run api
npm start
```

Open http://localhost:4200.
