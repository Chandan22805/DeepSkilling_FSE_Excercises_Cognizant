angular.json - Angular workspace build, serve and production configuration.
tsconfig.json - Base TypeScript compiler configuration.
tsconfig.app.json - TypeScript configuration for the application build.
package.json - Project dependencies and npm scripts.
src/main.ts - Application entry point that bootstraps the root Angular component.
src/app/app.config.ts - Standalone application providers for router, HTTP and NgRx.
src/app/app.component.ts - Root application component.
src/index.html - Host HTML page containing the Angular root element.
Production budgets warn above 500kB and error above 1MB to control bundle size.
The compiled application code is emitted by ng build into dist/student-course-portal/.
